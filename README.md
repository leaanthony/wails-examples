# wails-examples

NOTE: This project is a WIP and is probably not going to work for you yet.

Example projects that demonstrate the functionality of Wails v2.

## Menu

A simple program showing how to create menus.

Features:

  - [x] Application Menu
    - [x] Text
    - [x] Checkboxes
    - [x] Radio Groups
    - [x] Submenus
    - [ ] Images
    - [ ] Custom Fonts
    
  - [ ] Tray Menus
    - [ ] Icon
    
  - [ ] Context Menus
